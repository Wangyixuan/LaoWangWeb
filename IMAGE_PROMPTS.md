## 使用建议

1. **尺寸要求**：
   - Hero 主图：1920x1080 或更高
   - App 图标：1024x1024
   - 功能图标：512x512

2. **风格统一**：
   - 使用相同的配色方案（橙黄到琥珀渐变）
   - 保持现代简约风格
   - 确保所有图片风格一致

3. **格式**：
   - 推荐使用 PNG（带透明背景）或 WebP
   - 优化文件大小以提高加载速度

4. **工具推荐**：
   - Midjourney
   - DALL-E 3
   - Stable Diffusion

## 图片放置位置

生成的图片应放置在 `public/images/` 目录下：

```
public/
└── images/
    ├── appStoreLogo.png       # We Feel Logo (主图)
    ├── feature-emotion.png    # 情感表达图标
    ├── feature-community.png  # 社区互动图标
    ├── feature-privacy.png    # 隐私保护图标
    └── feature-language.png   # 多语言支持图标
```
